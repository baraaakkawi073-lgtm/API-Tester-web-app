# API Tester Web App

![Screenshot](1773548542324.jpg/1773548608657.jpg)

---

## Description

This is a simple **web-based API Tester** that allows you to:

- Enter any API URL
- Send requests (GET)
- Display the response in a readable format

It is built using **HTML, CSS, and JavaScript**, no backend required.

---

## Features

- Clean and responsive interface
- Works with JSON and plain text responses
- Displays results in formatted view
- Shows error messages if request fails

---

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript (ES6)
- Fetch API

---

## How to Use

1. Open `index.html` in your browser.
2. Enter the API URL in the input field.
3. Click the **Send Request** button.
4. View the formatted response in the result area.
